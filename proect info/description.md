# GhostGrid: AI-powered Digital Twin Platform

## 🌐 Overview
**GhostGrid** is a production-grade prototype for a Smart City Digital Twin focused on power infrastructure. It bridges the gap between physical electrical components and digital monitoring tools by creating a virtual, real-time mirror of the power grid.

## 🎯 Key Objectives
- **Grid Transparency:** Provide a "Single Source of Truth" for all grid assets.
- **Proactive Safety:** Use AI simulations to highlight high-risk zones before they cause outages.
- **Operational Efficiency:** Streamline data management for field engineers through a visual, map-first interface.

## 🌟 Visual Theme
The application features a **Dark Tech Theme** inspired by futuristic cockpits and power control centers. It utilizes vibrant neon accents to prioritize data visualization and clarify status indicators across the platform.

## 🏁 How to Run
1. Ensure Python 3.8+ is installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Launch the server: `python app.py`
4. Access the platform: `http://localhost:5000`
